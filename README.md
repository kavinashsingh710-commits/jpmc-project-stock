# Super Simple Stock Market

This code is a Python-based script for the Global Beverage Corporation Exchange (GBCE), as per the assignment requirements.

## Features

The model provides all required functionality:
* Calculates Dividend Yield for Common and Preferred stocks.
* Calculates P/E Ratio.
* Records trades with a timestamp, quantity, indicator, and price.
* Calculates a Volume Weighted Stock Price (VWSP) based on trades in the past 5 minutes.
* Calculates the GBCE All Share Index using the geometric mean of all stock VWSPs.
